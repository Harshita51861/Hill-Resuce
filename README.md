# Robot Coordination Maps

Human-in-the-Loop Rescue Monitoring Interface

## Project Overview

Robot Coordination Maps is a web-based visualization platform designed for monitoring and coordinating multiple robots in rescue or emergency environments.

The system provides an interactive map interface where operators can observe robot movement, track positions, and support coordination in real-time scenarios.

This project is part of a Human-in-the-Loop (HIL) rescue coordination system.

---

## Objectives

* Visualize multiple robots on an interactive map
* Support rescue mission coordination
* Provide real-time positional awareness
* Enable operator supervision

##  Installation & Setup

Clone the repository:

```
git clone https://github.com/your-username/robot-coordination-maps.git
```

Navigate into the folder:

```
cd robot-coordination-maps
```

Install dependencies:

```
npm install
```

Run the project:

```
npm start
```

The application will run at:

```
http://localhost:3000
```

---

## Important Note

The `node_modules/` folder is not included because it is automatically generated after running:

```
npm install
```
